# Customer Churn Prediction

Built an end-to-end Machine Learning project to predict customer churn.

## Features
- Data preprocessing
- Label Encoding
- Feature Scaling
- Model Training
- Streamlit Deployment
- Real-time Predictions

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Joblib

## Workflow

Data
↓
Preprocessing
↓
Model Training
↓
Save Model
↓
Streamlit Deployment
↓
Prediction

## Run Locally

pip install -r requirements.txt

streamlit run app.py
